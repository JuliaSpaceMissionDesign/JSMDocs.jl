    JSMDInterfaces.jl

    An interface package for the JSMD ecosystem

    SMDInterfacesUtils.jl handles all the interface definitions for the JSMD ecosystem. It is a low-level and low dependency library made to be depended on by the higher-level libraries to supply a common interface.
